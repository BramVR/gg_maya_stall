# Ship A Compact Maya Command Surface

Maya Stall v1 will ship a compact Maya-focused subset inspired by Crabbox: `init`, `doctor`, `run`, `status`, `stop`, `attach`, `screenshot`, `evidence collect`, and `evidence publish`. Recording remains deferred until the Session Broker exposes real recording capture. Broader Crabbox-style commands such as config mutation and host listing can come later when the workflow proves they are needed.
