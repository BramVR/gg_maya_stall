# Mirror Crabbox Visual Evidence Policy

Maya Stall runs must produce Visual Evidence using the same policy shape as Crabbox: screenshots are easy and expected, video/proof capture is available on demand or for failure/debug evidence, and a full Evidence Bundle can include screenshots, video, logs, metadata, scenes, and output files. Real Maya UI failures are too hard to diagnose from logs alone, so Visual Evidence is part of the core evidence model rather than optional polish.
