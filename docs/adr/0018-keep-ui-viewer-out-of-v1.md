# Keep UI Viewer Out Of V1

Maya Stall v1 will provide screenshot and recording capture but no built-in UI viewer. Recordings are desktop MP4 Visual Evidence captured through the Session Broker from the interactive Windows Maya session and encoded locally. Viewer support remains deferred until the Session Broker exposes stable proof surfaces. Crabbox includes VNC and WebVNC because it manages general desktop leases and a shared viewing portal; Maya Stall can rely on existing host viewing tools for now and add a separate viewer command later only when the Session Broker exposes a stable viewer path.
