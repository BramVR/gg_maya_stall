# Domain Docs

This repo uses a single-context domain layout.

## Read Order

- Read `CONTEXT.md` for Maya Stall domain language.
- Read relevant ADRs in `docs/adr/` before changing behavior.
- Read the PRD in `docs/prd/` when slicing or implementing v1 work.

## Layout

- Root context: `CONTEXT.md`
- Architectural decisions: `docs/adr/`
- Product requirements: `docs/prd/`

Use glossary terms from `CONTEXT.md` in issue titles, issue bodies, PRDs, docs, and code-facing names where practical.
