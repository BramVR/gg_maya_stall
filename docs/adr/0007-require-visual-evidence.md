# Mirror Crabbox Visual Evidence Policy

Maya Stall runs must produce Visual Evidence using the same policy shape as Crabbox where the Session Broker can prove it: screenshots are easy and expected, while recording is deferred for v1 until real broker-backed recording capture exists. A full Evidence Bundle can include screenshots, logs, metadata, scenes, and output files. Real Maya UI failures are too hard to diagnose from logs alone, so Visual Evidence is part of the core evidence model rather than optional polish.
