# Diagnose Host Prerequisites Instead Of Installing Them

Maya Stall v1 will follow Crabbox's static SSH model for owned hosts: it diagnoses host prerequisites but does not install or configure them. Maya Hosts must already provide OpenSSH, an interactive Windows desktop, Autodesk Maya, the Session Broker, writable work roots, and visual capture support; setup automation can be added later if the manual runbook becomes the bottleneck.
